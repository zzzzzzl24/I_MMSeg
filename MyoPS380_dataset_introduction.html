MyoPS380 Dataset Description

一、研究背景

评估心肌存活性对心肌梗死（MI）患者的诊断与管理具有重要意义。心脏磁共振成像（CMR）能够提供结构、功能与组织特征信息，三种常用序列具有互补性：

  • bSSFP：呈现清晰的心脏解剖与运动；
  • LGE：突出显示梗死及纤维化区域；
  • T2W：强调急性期损伤与水肿。

多模态联合可对心肌区域进行正常、瘢痕（Scar）、水肿（Edema）分区，但手工标注耗时且存在观察者差异。因此构建多模态自动分割模型具有重要价值。

二、数据来源及质量控制

在纳入 MyoPS380 数据集前，所有候选病例均经过系统化图像质量筛查，排除运动伪影、心动伪影、关键序列缺失及心室覆盖不足的病例。最终收集了 2017–2023 年仁济医院的 380 例多模态 CMR 数据，涵盖 bSSFP、LGE 与 T2W 模态。

三、数据组织结构

数据采用统一命名与文件结构，便于模型训练与复现：

MyoPS380_dataset/
    ├── Raw_data/
    │   ├── bSSFP/
    │   │     ├── case0001.nii.gz
    │   │     ├── ...
    │   │     └── case0380.nii.gz
    │   ├── LGE/
    │   │     ├── case0001.nii.gz
    │   │     ├── ...
    │   │     └── case0380.nii.gz
    │   ├── T2w/
    │   │     ├── case0001.nii.gz
    │   │     ├── ...
    │   │     └── case0380.nii.gz
    │   └── label/
    │         ├── case0001_gt.nii.gz
    │         ├── ...
    │         └── case0380_gt.nii.gz

其中 caseXXXX.nii.gz 表示患者单模态 3D 图像，caseXXXX_gt.nii.gz 为三类病变标注（正常心肌 / Scar / Edema）。所有文件均为 NIfTI 格式。

五、预处理流程

针对模态差异，本研究构建了结构化预处理流程：

(1) 切片级对应关系构建

利用 DICOM 的 ImagePositionPatient 计算各切片空间位置，基于欧氏距离建立多模态切片对应关系，剔除无法唯一匹配的切片。

(2) 空间标准化：重采样与中心裁剪

统一重采样至 1×1 mm 分辨率，中心裁剪至 224×224，并归一化至 [0, 255]。

(3) 基于心肌标签的二维仿射校正

利用心肌标签作为先验，引导逐切片仿射变换（旋转、缩放、剪切、平移），实现序列间初步对齐。

(4) Voxelmorph 非刚性深度配准

进一步利用 Voxelmorph 进行柔性变形校正，获得更高精度的跨模态配准。

(5) 最终 ROI 裁剪与封装

基于心肌 ROI 裁剪至 128×128，并将模态堆叠成 3D 体数据存储为 NIfTI。

六、伦理规范

MyoPS380 数据集已获得上海交通大学医学院附属仁济医院伦理委员会批准（批准号：[2018]093），并在发布前完成匿名化处理。
